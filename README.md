# 新春手势烟花玩法说明

这是一个基于手势识别的春节烟花页面，主文件是 `index.html`。

## 1. 玩法总览

- 打开页面后，如果暂时没有检测到手势，会先自动放一段暖场烟花。
- 暖场结束后，屏幕中间会提示：
  `请将手放入画面，五指张开并停住 0.5 秒开启表演`
- 演出开始后，可以通过不同手势控制普通烟花、超级烟花和结尾收官。

## 2. 手势控制

1. `✋ 五指张开并停住 0.5 秒`
- 开启表演（若已结束，也可再次开启新一轮）。
- 演出中保持五指张开，会触发全屏随机烟花。

2. `☝️ 单指（食指）`
- 手指指到哪里，哪里就升空普通烟花。
- 移动手指时，会沿着移动轨迹连续发射（已做限流，不会过密）。

3. `✌️ V 手势（超级烟花）`
- 进入 V 手势会先发射一枚超级烟花。
- 移动手势时会跟随发射超级烟花，但频率比单指更慢。
- 超级烟花体积更大、音效更低沉更响亮。

4. `✊ 握拳约 1 秒`
- 结束当前演出并触发收官流程。

## 3. 收官流程（自动）

1. 从左到右依次升空一批收官烟花  
2. 爆炸后用星光汇聚出“马”  
3. 马的上方显示“新春快乐”  
4. 最后切到对联收尾（含横批、福字、左右联）并出现红包雨

## 4. 体验细节

- 背景为纯黑夜空，烟花残影会自然衰减，不会长期留色。
- 手离开画面时：
  - 如果表演进行中，系统会继续自动放烟花，不会中断。
  - 如果尚未开始，会继续显示开场提示。
- 屏幕上会显示手势定位点，便于对准发射位置。

## 5. 使用建议

- 建议在光线充足、背景相对干净的环境下使用摄像头识别手势。
- 手势切换时停留约 `0.2 ~ 0.5` 秒，识别会更稳定。
- 若声音未播放，先与页面交互（让浏览器解锁音频上下文）。

## 6. 快速开始

1. 进入项目目录：

```bash
cd spring-festival-fireworks
```

2. 用浏览器打开：

- 直接双击 `index.html`，或
- 使用本地静态服务打开（可选）。
